
# Vercel部署指南

## 基本部署步骤

1. **注册Vercel账号**
   - 访问 [Vercel官网](https://vercel.com) 并创建账号
   - 可以通过GitHub、GitLab或Bitbucket账号直接登录

2. **导入项目**
   - 登录Vercel后，点击"Add New..."，然后选择"Project"
   - 从列表中选择您的Git仓库
   - 如果没有连接Git仓库，点击"Import Git Repository"并关联您的代码仓库

3. **配置项目**
   - 部署时使用默认设置即可，Vercel会自动检测您使用的是React+Vite项目
   - 构建命令应该已设为 `npm run build` 或 `vite build`
   - 输出目录应该已设为 `dist`

4. **部署项目**
   - 点击"Deploy"按钮开始部署
   - 部署完成后，Vercel会提供一个默认域名（例如：your-project.vercel.app）

## 确保项目公开可访问

Vercel上的项目**默认是公开可访问的**，无需特别设置。但请确认以下几点：

1. **检查项目权限**
   - 进入项目设置 → 一般设置
   - 确保未开启"Password Protection"（密码保护）选项

2. **检查域名设置**
   - 进入项目设置 → 域名
   - 确认默认的Vercel域名（*.vercel.app）已激活且可访问

3. **团队项目设置**
   - 如果您使用的是团队账户，确保项目的访问级别设置为"Public"
   - 进入团队设置 → 项目 → 选择您的项目 → 访问设置

## 测试访问

部署完成后：
1. 使用匿名浏览器窗口（隐身模式）访问您的Vercel应用URL
2. 确保未登录状态下可以正常访问应用
3. 请朋友或同事在不同设备上测试访问

## 常见问题

- **如果应用仍然无法访问**：
  - 检查应用的路由配置是否正确
  - 确保index.html正确加载资源（本项目已配置使用相对路径加载资源）
  - 查看Vercel部署日志，检查是否有构建或部署错误

- **如果需要更改域名**：
  - 在项目设置中添加自定义域名
  - 按照Vercel提供的DNS配置说明设置您的域名
